# GERADOR DE SENHAS
App para gerar senhas de acordo com as configurações selecionadas.

## Opções disponíveis
- Tamanho da senha (caracteres)
- Letras minúsculas
- Letras maiúsculas
- Números
- Símbolos
- Espaços
- Caracteres duplicados
